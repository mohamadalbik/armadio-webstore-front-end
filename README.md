# React Webstore Frontend

A modern e-commerce frontend built with React and Bootstrap. This project provides the user interface for a webstore with a clean design and smooth user experience.

## Features

- Product detail pages
- Shopping cart interface
- Search and filter functionality
- Clean and modern UI design

## How to Run

### Prerequisites

- Node.js (14.x or higher)
- npm

### Run the Application

1. **Clone this repository:**
   ```
   git clone https://github.com/mohamadalbik/armadio-webstore-front-end.git
   cd react-webstore
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Start the development server:**
   ```
   npm start
   ```

4. **Open your browser and go to:**
   ```
   http://localhost:3000
   ```
